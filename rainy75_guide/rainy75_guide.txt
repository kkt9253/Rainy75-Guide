========================================
Rainy75 키보드 사용법 종합 가이드
========================================

** 이 문서는 Rainy75 키보드의 모든 사용법을 정리한 가이드입니다. **


[ 1. 초기 설정 ]
----------------------------------------
- 전원 스위치 위치: Caps Lock 키캡 아래 (무선 사용 시 ON, 유선 사용 시 OFF 권장)


[ 2. 연결 모드 전환 (FN + Tab) ]
----------------------------------------
- FN + Tab 키를 반복해서 눌러 유선 <-> 2.4GHz <-> 블루투스 모드로 순환 전환합니다.
- 현재 모드는 키보드 상단 LED 색상으로 확인 가능합니다. (ESC 또는 F1~F4)

  (1) 유선 모드
      - 케이블을 PC와 연결합니다.
      - FN + Tab을 눌러 ESC 키에 LED가 들어오면 유선 모드입니다.

  (2) 2.4GHz 무선 모드
      - USB 동글을 PC에 연결하고, 키보드 전원을 ON 합니다.
      - FN + Tab을 눌러 F4 키에 LED가 들어오면 2.4GHz 모드입니다.
      - ※ 연결 불량 시: F4가 느리게 깜빡일 때, FN + F4를 3초간 길게 눌러 페어링 모드(빠르게 깜빡임)로 진입 후 다시 시도합니다.

  (3) 블루투스 모드
      - 키보드 전원을 ON 합니다.
      - FN + F1 / F2 / F3 중 하나를 짧게 눌러 원하는 슬롯을 선택합니다.
      - 선택한 키(예: F1)를 FN과 함께 3초간 길게 누르면 페어링 모드(빠르게 깜빡임)가 됩니다.
      - 연결할 기기(PC, 스마트폰)의 블루투스 설정에서 'Rainy75-1' 등을 찾아 연결합니다.
      - ※ 기기 전환: FN + F1 / F2 / F3 짧게 누르기


[ 3. RGB 백라이트 제어 (자주 사용!) ]
----------------------------------------
- FN + Backspace       : 백라이트 ON / OFF
- FN + ↑ (방향키)      : 밝기 증가
- FN + ↓ (방향키)      : 밝기 감소
- FN + ← (방향키)      : 효과 속도 감소
- FN + → (방향키)      : 효과 속도 증가
- FN + \ (백슬래시)    : 조명 효과 모드 변경 (순환)
- FN + Enter           : 조명 색상 변경


[ 4. 주요 기능 및 단축키 ]
----------------------------------------
- FN + Space (3초)     : 배터리 잔량 확인
                         (RGB 모델: 숫자 1~0 키에 초록색으로 표시, 1=10% ... 0=100%)
                         (Non-RGB 모델: ESC~F4 키에 표시)

- FN + M (3초)         : Windows <-> Mac 모드 전환
- FN + Win             : Windows 키 잠금 / 해제

- FN + H               : 초저지연 모드 (유선 연결 시 사용, Caps Lock이 깜빡이며 상태 표시)
- FN + L               : 저전력 모드 (배터리 절약, 최신 펌웨어)

- FN + Page Up         : Print Screen
- FN + Page Down       : Pause
- FN + Home            : Scroll Lock
- FN + End             : Insert


[ 5. 배터리 상태 표시 ]
----------------------------------------
- 배터리 부족 시: ESC 키가 빨간색으로 계속 깜빡입니다.
- 충전 중: ESC 키가 숨쉬기 모드로 빛납니다. (펌웨어 버전에 따라 다름)
- 완충 시: ESC 키가 녹색으로 바뀌거나, 원래 조명 색으로 돌아옵니다.


[ 6. 기타 유용한 정보 ]
----------------------------------------
- 공장 초기화 (Factory Reset):
  만약 키보드 작동에 문제가 생겼을 경우, FN + ESC 키를 5초 이상 길게 눌러 공장 초기화를 시도해 볼 수 있습니다. (키맵핑 설정 등은 초기화됩니다.)

- VIA 키맵핑 변경:
  'wobkey.com' 공식 홈페이지의 Download Center에서 키보드에 맞는 JSON 파일을 받아 'usevia.app' 웹사이트에서 키를 커스터마이징할 수 있습니다.
  (자세한 방법은 별도 가이드 참고 필요)
